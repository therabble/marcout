Web Service deploy checklist

1. Install Python :
    Debian and Ubuntu:
        `sudo apt install python-3`

2. Python virtual environment (for running a specific Python and Python
    application installs in an isolated directory):
    Included in python 3. 

    Available in Python 3.3+;
    Recommended as of Python 3.5:
    `venv`
    `python3 -m venv /path/to/new/virtual/environment`
    (https://docs.python.org/3/library/venv.html)


3. Set up virtual environment

4. In virtual environment, install pip 3 Python package manager

5. In virtual environment, install Flask web server

6. In virtual environment, create Flask server instance
